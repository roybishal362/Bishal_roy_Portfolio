# Bishal Roy — AI Engineer Portfolio 🚀

![Portfolio Preview](./public/bishal.png) <!-- Update with actual preview screenshot if added later -->

A high-performance, responsive AI Engineering portfolio built with **React**, **Vite**, **TypeScript**, **GSAP**, and **Framer Motion**. Features a bespoke "Dark Glass Cosmos" design system with neural-network canvas animations, glassmorphism bento grids, and smooth Lenis scroll.

🌐 **Live Demo:** [Deploying Soon...]

---

## ✨ Features

- **Dark Glass Cosmos Design System:** Void black (`#050508`) background with electric indigo (`#6366f1`) and cyan (`#22d3ee`) glows.
- **Neural Network Canvas:** Custom HTML5 Canvas hero animation featuring interactive, connecting data nodes.
- **Glassmorphism Bento Grid:** Modern achievement showcase with hovering gradient borders.
- **Smooth Scrolling & GSAP Animations:** Hardware-accelerated scroll reveals, animated number counters, and staggered timeline animations powered by Lenis and GSAP ScrollTrigger.
- **Interactive Side Panels:** Slide-in modal overlays for deep dives into projects, research papers, and hackathons.
- **Responsive Typography:** Premium font stack featuring **Space Grotesk** (headlines), **Inter** (body), and **JetBrains Mono** (data/metrics).

## 🛠️ Tech Stack

- **Framework:** React 18, Vite
- **Language:** TypeScript
- **Styling:** Custom CSS3 with CSS Variables (No Tailwind)
- **Animation:** GSAP, ScrollTrigger, Framer Motion
- **Scroll Engine:** `@studio-freight/lenis`
- **Deployment:** Vercel (Recommended)

## 📁 Key Sections

- **Hero:** Impactful introduction with animated metrics (AIR 2, Top 8%, 5M+ Records).
- **Projects:** Selected work (C-TRUST, Cricket Scouting Engine, SHL RAG Pipeline) with full tech stacks and performance metrics.
- **Experience:** Vertical glowing timeline of ML/AI Internships.
- **Hackathons & Research:** Podium-style competition cards and academic publication status.
- **Tech Stack Marquee:** Infinite scrolling skills marquee featuring industry-standard AI/ML tooling.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/roybishal362/bishal-roy-ai-engineer-portfolio.git
   cd bishal-roy-ai-engineer-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
